# DND PWA Map

Интерактивная карта мира для D&D как PWA-приложение. Первая версия работает без сервера: данные сохраняются в браузере. Есть экспорт/импорт JSON.

## Локальный запуск

```bash
npm install
npm run dev
```

Откройте адрес из терминала, обычно http://localhost:5173

## Сборка

```bash
npm run build
npm run preview
```

## Деплой на Vercel

1. Загрузите проект в GitHub.
2. На vercel.com нажмите Add New Project.
3. Выберите репозиторий.
4. Framework: Vite.
5. Build command: `npm run build`.
6. Output directory: `dist`.
7. Deploy.

## Как заменить карту

Замените `public/sample-map.svg` на свою картинку, например `world-map.jpg`, и в `src/main.jsx` измените:

```js
mapUrl: '/world-map.jpg'
```

Если пропорции карты другие, поменяйте bounds:

```js
bounds: [[0,0],[1000,1400]]
```

где 1000 — высота картинки, 1400 — ширина.

## Ограничение первой версии

Данные не синхронизируются между игроками автоматически. Для обмена используйте Экспорт/Импорт JSON. Следующий этап — подключить Supabase, чтобы у всех была общая карта.
